# rudiak-kirill-site

Личный сайт Кирилла Рудяка на GitHub Pages.

Репозиторий ведётся как самостоятельный внешний ресурс внутри рабочей области `cursor-repos`.

## Ссылки

- Сайт: https://rudiak-kirill.github.io
- GitHub: https://github.com/Rudiak-Kirill/rudiak-kirill.github.io
- Рабочая область: [cursor-repos](../)
- Связанный проект: [SEO Risk Check](../seorisk-repo/)

## Назначение

- Публичная витрина SEO-специалиста.
- Статические страницы: HTML, CSS, JavaScript.
- Размещение через GitHub Pages.

## Работа с репозиторием

```bash
git status
git pull
```

Коммиты и деплой проверять из папки `rudiak-kirill-site`, а не из корня рабочей области.
